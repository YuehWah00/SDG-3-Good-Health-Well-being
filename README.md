# SDG 3: Good Health & Well-Being (Malaysia)

## Project Description

This project is a responsive informational website designed to promote **Sustainable Development Goal 3 (SDG 3): Good Health & Well-Being**, with a specific focus on the Malaysian context. The website aims to raise awareness about health issues, encourage healthy lifestyles, and provide accessible information through visual content, wellness tips, news updates, and data visualizations.

The site combines **educational content**, **visual storytelling**, and **user-friendly interaction** to support public understanding of physical health, mental well-being, and preventive healthcare. It aligns with SDG 3 objectives such as disease prevention, health promotion, and improved access to health knowledge.

### Key Objectives
- Educate users about SDG 3 and Malaysia’s health challenges  
- Promote healthy daily habits and mental well-being  
- Present health data and statistics in a clear visual format  
- Encourage public engagement through accessible design and contact features  

---

## Features

- **Hero Section** with strong visual messaging and call-to-action  
- **About Section** explaining SDG 3, Malaysia’s health overview, and project mission  
- **Wellness Tips Section** covering:
  - Healthy Eating  
  - Exercise & Fitness  
  - Mental Health Care  
  - Daily Healthy Habits  
- **News Section** with real external links to health-related articles  
- **Infographics Gallery** with lightbox-style image viewing  
- **Statistics & Data Visuals** using charts and explanatory text  
- **Contact Form** with file upload and basic form validation  
- **Responsive Navigation** with mobile toggle menu  
- **Footer** with SDG branding and social media links  

---

## Technologies Used

- **HTML5** – Semantic structure and content layout  
- **CSS3** – Layout, typography, color system, responsiveness  
- **JavaScript (jQuery)** – Interactivity and UI behaviors

## UI Style Guide

### 1. Design Concept
The visual design emphasizes **cleanliness, calmness, and trust**, reflecting healthcare and well-being themes. A **green-and-white color palette** is used to symbolize health, growth, and sustainability, while soft shadows and rounded corners create a friendly, approachable interface.

---

### 2. Color Palette

| Purpose | Color |
|------|------|
| Primary Green | `#4CAF50` |
| Light Green Background | `#e8f5e9` |
| Dark Green Hover | `#43a047` |
| Text (Primary) | `#222222` |
| Background (Light) | `#f5f7f6` |
| White | `#ffffff` |

---

### 3. Typography

- **Primary Font:** Arial, sans-serif  
- **Heading Style:** Bold, large-size for clarity and hierarchy  
- **Body Text:** Regular weight with comfortable line-height (1.5–1.7)  
- **Emphasis:** Used sparingly through headings and underlined insights  

---

### 4. Layout & Spacing

- **Max content width:** 1100px (centered layout)  
- **Section spacing:** 60px vertical padding  
- **Grid systems:** CSS Grid for:
  - News cards  
  - Posters  
  - Statistics rows  
- **Consistent margins and padding** for readability  

---

### 5. Components

#### Buttons
- Rounded corners  
- Green background  
- White text  
- Hover color change for feedback  

#### Cards
- White background  
- Soft shadow  
- Rounded corners  
- Hover lift / scale animation  

#### Images
- Fully responsive  
- `object-fit: cover` for consistent visuals  
- Rounded edges for a modern look  

---

### 6. Interaction & Motion

- Hover effects on cards and posters  
- Light zoom effect on social media icons  
- Smooth scaling transitions (CSS-based)  
- JavaScript-enhanced interactions:
  - Mobile navigation toggle  
  - File upload name display  
  - Contact form submission feedback  

---

### 7. Responsiveness

The website is fully responsive and adapts to different screen sizes:
- Navigation collapses on mobile  
- Single-column layouts for wellness, posters, and statistics  
- Scaled typography for smaller screens  
- Touch-friendly buttons and spacing  

---

## JavaScript Functionality Summary

- Enhances hover effects for cards  
- Displays selected file name in contact form  
- Prevents default form submission (demo behavior)  
- Displays confirmation alert on form submission  
- Toggles navigation links on mobile screens
